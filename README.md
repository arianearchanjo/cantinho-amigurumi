# Cantinho do Amigurumi 🧶

Site institucional para ateliê de crochê artesanal, desenvolvido com HTML, CSS e JavaScript puro.

🔗 **[cantinho-amigurumi.netlify.app](https://cantinho-amigurumi.netlify.app/)**

---

## Sobre o projeto

O Cantinho do Amigurumi é um ateliê artesanal de Campina Grande do Sul, PR, especializado em amigurumis feitos à mão sob encomenda. O site foi desenvolvido para apresentar as coleções, facilitar o contato via WhatsApp e transmitir a identidade acolhedora e artesanal da marca.

## Páginas

- **Home** — hero editorial, teaser de coleções, seção biscuit, processo e CTA
- **Produtos** — catálogo completo com 6 coleções e navegação por categoria
- **Galeria** — grid masonry com filtros e lightbox
- **Sobre** — história da artesã

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis customizadas e design system próprio
- JavaScript vanilla modular
- Bootstrap 5 (grid)
- Lucide Icons
- Google Fonts — Playfair Display, Cormorant Garamond, DM Sans, Caveat
- Deploy via Netlify

## Funcionalidades

- Design responsivo para mobile e desktop
- Navbar sticky com menu hamburguer
- Marquee animado
- Scroll reveal com Intersection Observer
- Grid masonry com filtros por categoria
- Lightbox para galeria
- Barra de progresso de scroll
- Integração com WhatsApp
- Parallax sutil no hero

## Deploy

O projeto é hospedado no [Netlify](https://netlify.com) sem domínio próprio por opção do cliente.
